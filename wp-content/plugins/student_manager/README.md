# Student Manager

Plugin quản lý sinh viên cho WordPress.

## 📝 Giới thiệu
Plugin "Student Manager" giúp quản trị viên quản lý danh sách sinh viên, lưu trữ thông tin và hiển thị ra frontend dưới dạng bảng.

## ⚙️ Cài đặt
1. Tải thư mục `student-manager` vào `wp-content/plugins/`.
2. Kích hoạt plugin trong trang quản trị WordPress.

## 🚀 Sử dụng
- Vào menu **Sinh viên** để thêm mới, chỉnh sửa thông tin sinh viên.
- Các trường thông tin:
  - **Họ tên** (title)
  - **Tiểu sử/Ghi chú** (editor)
  - **MSSV** (mã số sinh viên)
  - **Lớp/Chuyên ngành** (CNTT, Kinh tế, Marketing)
  - **Ngày sinh**

### Hiển thị danh sách sinh viên
Chèn shortcode sau vào bất kỳ Page nào:

```
[danh_sach_sinh_vien]
```

Kết quả sẽ hiển thị bảng gồm: STT, MSSV, Họ tên, Lớp, Ngày sinh.

## 🎨 Tuỳ chỉnh giao diện
- Có thể chỉnh sửa file `assets/style.css` để thay đổi giao diện bảng hiển thị.

## 📸 Ảnh minh hoạ
> Thêm ảnh chụp màn hình tại đây (ví dụ: ảnh bảng sinh viên trên frontend, ảnh trang thêm sinh viên).
